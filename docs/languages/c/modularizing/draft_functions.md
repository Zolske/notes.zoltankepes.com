## Memory Manipulation

#### memcpy `void *memcpy(void *dest, const void *src, size_t numBytes);` `<string.h>`

Is used to copy a block of memory from one location to another. Unlike other copy functions, the memcpy function copies the specified number of bytes from one memory location to the other memory location regardless of the type of data stored.

#### memset `void *memset(void *str, int c, size_t n)`

Copies the character c (an unsigned char) to the first n characters of the string pointed to, by the argument str.

- str âˆ’ This is a pointer to the block of memory to fill.
- c âˆ’ This is the value to be set. The value is passed as an int, but the function fills the block of memory using the unsigned char conversion of this value.
- n âˆ’ This is the number of bytes to be set to the value.

##### RETURN VALUE

This function returns a pointer to the memory area str.

##### example for an int

```c
#include <stdio.h>
#include <string.h>

void printArray(int arr[], int n)
{
   for (int i=0; i<n; i++)
      printf("%d ", arr[i]);
}

int main()
{
    int n = 10;
    int arr[n];

    // Fill whole array with 100.
    memset(arr, 10, n*sizeof(arr[0]));
    printf("Array after memset()\n");
    printArray(arr, n);

    return 0;
}
```
